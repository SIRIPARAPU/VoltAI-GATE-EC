## GATE AI Command Center v2

A simple, topic-aware study app for **GATE EC**:

- **Homepage**: subject cards
- **Subject page**:
  - **Theory**: static lecture list (no runtime playlist fetching)
  - **Practice**: topic list (strictly from syllabus data)
- **Revision notes** + **Practice engine**: generated by AI with strict prompts
- **KaTeX**: renders formulas
- **Global countdown timer**: same everywhere (LocalStorage)

### Run locally

```bash
cd gate-ai-command-center-v2
npm run dev
```

Open `http://localhost:3000`.

### One-time playlist sync (already done)

This project stores playlist video IDs in `src/data/playlistVideoIds.json` to avoid runtime fetching bugs.

If you ever want to refresh them:

```bash
npm run sync:playlists
```

### AI key (required)

Go to **Settings** in the app and paste your API key. It is stored in your browser’s LocalStorage (for a personal app).
